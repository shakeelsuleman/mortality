# UK Covid cases - animation
## This short script retrieves public Covid-19 mortality data and plots an animation of trends


* [ECDC Website](https://www.ecdc.europa.eu/en/publications-data/download-todays-data-geographic-distribution-covid-19-cases-worldwide)

* [WHO Covid19 Situation Reports](https://www.who.int/emergencies/diseases/novel-coronavirus-2019/situation-reports)

* [Covid-19 Guiance for health professionals](https://www.gov.uk/government/collections/wuhan-novel-coronavirus)

* [COVID-19: track coronavirus cases](https://www.gov.uk/government/publications/covid-19-track-coronavirus-cases)

* [Date Source: UK Government data](https://api.coronavirus.data.gov.uk/v2/data?areaType=overview&metric=newDeaths28DaysByDeathDate&format=csv)
